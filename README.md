# s_ch-OCR 项目介绍s_ch-OCR 是一款轻量级光学字符识别（OCR）工具，以极简架构实现高效的字符识别能力，无冗余模块，部署和运行成本低。
核心文件说明
app.py：项目核心逻辑载体，集成 OCR 全流程能力，包括图像预处理、字符检测、识别算法执行及结果输出等核心功能，是整个工具的核心执行单元。
run.bat：Windows 系统快捷启动脚本，一键触发 app.py 执行，简化手动运行指令的操作（需提前配置 Python 及 OCR 基础依赖）。
核心架构
项目采用两层极简架构，无复杂层级依赖：
核心逻辑层（app.py）：承载 OCR 核心算法与业务逻辑，完成从图像输入到识别结果输出的全流程；
快捷启动层（run.bat）：作为入口层，仅负责快速调用核心逻辑层，降低用户使用门槛。
整体架构轻量化，聚焦 OCR 核心能力，适配快速部署和简单场景下的字符识别需求。# s_ch-OCR 项目介绍s_ch-OCR 是一款轻量级光学字符识别（OCR）工具，以极简架构实现高效的字符识别能力，无冗余模块，部署和运行成本低。
